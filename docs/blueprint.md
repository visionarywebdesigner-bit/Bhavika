# **App Name**: Bhavika's Heart

## Core Features:

- Boot-Up Screen: Display a full-screen background gradient with a central heart animation and loading indicator, setting a romantic and gentle tone.
- Interactive Heart Transition: Upon tapping the heart, initiate a transition with a soft blur and a chime sound, leading into the name reveal sequence.
- Floating Name Hearts Animation: Animate hearts containing the name 'Bhavika' floating upwards, creating an emotional and magical transition to the next page.
- Gentle Page Transition: Create a smooth transition to the next page with a lavender-pink gradient and a soft fade-in of the 'Welcome, Bhavika' message.
- Personalized Welcome Message Generation: Generate personalized welcome messages that appear upon transitioning to the main content, tailoring the experience to the user. This requires using an LLM tool to modify various properties of the message such as sentiment, length, etc.

## Style Guidelines:

- Primary color: Soft pink (#F9CACA) to evoke romance and tenderness.
- Background color: Desaturated baby pink (#FDE2E2) to provide a warm and gentle backdrop.
- Accent color: Lavender (#E6E6FA) for a touch of dreamy elegance and contrast.
- Font for names/messages: 'Belleza' serif to bring a stylish handwritten feel. For any additional body text, use 'Alegreya' serif to bring an elegant intellectual look
